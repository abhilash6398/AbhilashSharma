HACCP Management is core to the HACCP Now software system. It provides tried and tested HACCP tools that help users produce internationally compliant HACCP documentation quickly and effectively.

The HACCP plan module provides a logical and structured approach to the documentation of:

  • Terms of Reference
  • Process Flow Diagram
  • Hazard Analysis

The system’s built in knowledge database of hazards, their controls, monitoring and corrective actions helps carry out hazard analysis in a fast, consistent way.

Template hazard analysis can be created and then linked with process steps in multiple process flow diagrams. This means that when you are developing a new product, most of the hazard analysis can be done in minutes rather than days.

It also means that when hazard analysis is carried out, all affected HACCP studies are automatically updated.

The HACCP plan can be created as PDF documents instantly and may be emailed with configurable options to customers or auditors who request it within seconds.

The HACCP plan module can also be linked up to documents in almost any location and then published as a set of web pages on the web, or on your company network, for staff to browse at their leisure.

Monitoring procedures in each process step can be linked to electronic data capture forms so that the documentation is both integrated and relevant.

HACCP Now has a built-in decision tree based on Codex Alimentarius guidelines. It also features a configurable risk assessment tool that makes the process of identifying significant hazards logical and straightforward. Auditors are regularly impressed with HACCP Now’s capability and the standard of documentation produced by HACCP Now.
